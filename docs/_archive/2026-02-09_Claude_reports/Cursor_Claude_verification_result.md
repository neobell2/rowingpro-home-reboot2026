## 1. 이해되지 않는 지점

### 1.1 "병렬 운용" 개념
- 외부인은 "포트폴리오 다각화"로 오해 가능
- 생존을 위한 필수 구조임이 명확히 전달되지 않음

### 1.2 "생태계" 표현
- 추상적
- 구체적 연결 구조 불명확

### 1.3 "1보트 1코치 동승"
- 의미: 레저용 4X 보트에 1~4명 탑승 시 코치 1명 필수 탑승
- 문제: 설명 없음
- 외부인은 이 구조를 모름
- 체험업 대비 차별화 근거이나 활용 안 됨

### 1.4 창업주 직함 변경 설명
- "겸업금지"는 내부 사정
- 외부는 이해 불가하고 관심 없음
- BI에서 제외 권장

### 1.5 협회 "주관" vs "운영"
- 법적 관계 불명확
- 공신력 근거로 사용 시 정확한 표현 필요

## 2. BI 위험 요소

### 2.1 방어적 표현 (중대)
| 표현 | 위험 |
|---|---|
| "비인기 종목" | 자기 비하 |
| "가격 안 쌈" | 방어적, 정당화 실패 |
| "쉽지 않음" | 진입장벽 과도 강조 |
| "모든 사람 타겟 아님" | 배타적 인상 |

→ 내부 정리용으로는 정확하나 외부 커뮤니케이션 부적합

### 2.2 "전문 집단" 표현
- 비영리/동호회 연상
- 법인격 불명확
- B2B 타겟은 "회사/업체"와 거래 희망
- "전문회사" 원칙 확정 (맥락별 기업, 집단 허용)

### 2.3 창업주 1인 구조 과다 노출
- 솔직함은 맞으나 노출 수위 조절 필요
- '전문가 네트워크 구조'로 재프레이밍 가능

### 2.4 "체험 아님" 반복
- 부정 프레임 누적
- 타겟별 언어 분리로 해결

### 2.5 내부 제약 외부 전달
- 겸업금지, 직함 변경 등 내부 사정 노출
- BI에서 제외 필수

### 2.6 "1보트 1코치" 미활용
- 강력한 차별화 요소
- 체험업은 강사가 육지에서 지시
- 로잉프로는 코치가 보트 탑승
- 가격 정당화 + 안전 증명 핵심 근거

## 3. 정체성 충돌 가능성

### 3.1 체험 vs 전문 이분법
**충돌:**
- "체험이 아니다" 선언
- "가족 체험 수요" 타겟 설정

**원인:**
- 단일 메시지로 모든 타겟 대응 시도

**해결 방향:**
- 타겟별 언어 완전 분리
- B2B: "전문 스포츠"
- 가족: "전문가가 운영하는 체험"

### 3.2 조정 전문가 vs 생계 지원
**모호:**
- "전문 집단"인가 "생계 구조"인가

**해결 방향:**
- "전문가가 운영하는 전문회사"로 명확화
- 생계 지원은 내부 목표, 외부 메시지 아님
