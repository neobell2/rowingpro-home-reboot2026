### 4. 프로그램 구조 완전 이해 ✅

#### Indoor Program (실내)
- **3시간 M1/M2/M3 구조**
- 에르고미터 기반
- 개인전+단체전+시상
- 파워머니 보상 시스템
- 암웨이 프로그램 (2016) 사례 분석

#### Outdoor Program (수상)
- 보트 중심, 미사리 경기장
- 선착장/경기장 운영
- **선수 출신 무조건 동승** (안전)
- 미사리 = 서울 인근 유일 수상 자원

#### MIX Program (조정+팀빌딩)
- POST 1: Rowing (보트 또는 에르고)
- POST 2-6: 팀빌딩 액티비티 (명랑운동회, 워크샵, GX 등)
- 순환식 스테이션 운영
- 대규모(240명) 가능
- 현대차 "Vision Sculling" (2016) 사례 분석
- 파트너: 팀플레이 (우호협력회사)

#### Premium Outdoor
- C-level 소규모 (8-15명)
- 깊은 몰입, 고급 케이터링
- 다루기 어려운 시장
