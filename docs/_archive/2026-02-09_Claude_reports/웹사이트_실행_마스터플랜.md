# 4. 필수 운영 규칙

-   Home 완성 전 서브페이지 제작 금지
-   IDE 직접 수정 최소화
-   모델 역할 고정 (**Gemini = 생산 / Claude = 품질**)
-   구조 확보 후 디테일 개선

------------------------------------------------------------------------

# 5. 크리티컬 경로 (순서 고정)

    → 템플릿화
    → 페이지 양산
    → 기술 검수
    → 배포
