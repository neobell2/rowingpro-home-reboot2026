# 프로젝트 검수기록 (최종본 / 문서 미반영)

## 1. 검수 전제

* 본 기록은 **검수관 관점의 사실 기록**입니다.
* **추론·보완 작성 없음**, 근거 없는 판단 없음.
* 문서 수정은 포함하지 않으며, **기록 목적**입니다.

---

## 2. 있었어야 하나 누락된 항목 (확정)

### 2.1 IA / 페이지 구조

* **Privacy 페이지**

  * Footer에 `/privacy` 링크 요구 존재
  * IA(11페이지)에 페이지 정의 없음
  * 상태: **누락 (확정)**
* **404 페이지**

  * DEV_SPEC에 톤 규칙 존재
  * ROADMAP/구현 항목에 생성 작업 없음
  * 상태: **누락 (확정)**

---

## 3. 있었으나 통합 과정에서 소실된 항목 (클로드 정리 과정 오류)

* **SEO 실체 산출물**

  * sitemap.xml: 개념 언급 없음, 파일 단위 정의 없음
  * robots.txt: “기본 index, follow” 문구만 존재
  * canonical 정책: 기준 URL/정규화 규칙 없음
  * 상태: **통합 과정에서 실무 항목 탈락**
* **도메인 정규화 정책**

  * 도메인 존재하나 www/루트/슬래시 기준 없음
  * 상태: **정책 레벨 문서화 누락**

---

## 4. 모호하거나 정의 불충분 (판단 불가)

* **Programs 페이지**

  * 목적 정의 없음
  * Service와의 관계 불명
* **Service 허브 페이지**

  * 하위 템플릿은 명확
  * 허브 카드 구성 규칙 없음
* **OG 이미지 정책**

  * 히어로 우선 규칙만 존재
  * 전용 OG 제작 여부 불명

---

## 5. 원래 문서에 없었고, 현재도 없음 (미정 영역)

> 아래 항목은 **문서 어디에도 근거가 없어 판단 불가**
> (검수관은 추론하지 않음)

* Contact 폼 데이터 저장 위치
* 문의 수신 주체 및 알림 방식
* 스팸 방지 유무
* 개인정보 처리방침과 폼의 법적 연결 방식
* Analytics 도입 여부
* 이미지 최적화(webp/avif, 용량 기준)
* Cloudflare Pages 빌드/환경변수/리다이렉트 규칙

상태: **미정 / 확인 불가**
