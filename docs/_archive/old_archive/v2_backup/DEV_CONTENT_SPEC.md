## 6. 조정(Rowing) 종목 정의

### 불변 정의
```
Rowing = Teamwork
```
- 이 문장은 슬로건이 아니라 **정의(definition)**
- 조정을 단 한 문장으로 설명해야 한다면, 다른 답은 존재하지 않음

### 다른 단체종목과의 차이
- **스타 플레이어가 존재하지 않음**
- 한 명의 개인 기량으로 경기를 뒤집을 수 없음
- 개인의 탁월함은 팀에 흡수되거나, 팀을 무너뜨림
- **팀워크가 없으면 경기 자체가 성립하지 않음**

### '뒤로 가며 앞으로 전진'
- 조정 선수는 진행 방향을 보지 못한 채 노를 젓음
- 앞을 보는 사람은 단 한 명, **콕스(Coxswain)**뿐
- 팀의 방향·속도·판단은 모두 콕스에게 위임

### 신뢰 구조
- 로워는 보지 못한 채 실행
- 콕스는 믿지 않으면 판단할 수 없음
- **조정에서 신뢰는 감정이 아니라 기능**

## 7. '왜 조정인가' 실행 기준

### 본질 기준
- 조정은 극기·계발·건강이 아니라 **목적을 가진 놀이 구조**
- 즐거움·유희·쾌락은 삭제 대상이 아니라 **보존 대상**
- 완결을 향한 밀어넣기가 사라지면, 그 순간 조정은 다른 무엇이 된다

### 인간 기준
- 페르소나는 직무·연령·성별이 아니라 **놀이에 진입한 상태**로 판단
- 혼자 완결될 수 없다는 전제를 숨기지 않는다
- 개인의 쾌락은 집단의 완결에 종속되는 구조를 약화시키지 않는다

### 표현 기준
- 설명은 최소, 장면이 우선
- 말로 납득시키려는 순간, 잘못된 방향
- 이미지는 **"보는 즉시 이해되는가"**만 통과 기준

## 8. 페르소나 / 노출 순서

### 1차 필터: '막내'
- 상단에서 안전·정식·관리 가능성만 확인
- "무리 없는지 / 사고 안 나는지 / 설명 가능한지"

### 강도 노출
- **선택 진입 이후** 노출
- 상단은 통과 신호 중심

### C레벨 인식
- **"빡세지만 괴롭지 않음"**을 동시에 인식
- 놀이 구조는 유지하되, 관리되는 스포츠

## 9. 이미지 장면 기준 (3슬롯)

### hero (히어로)
- **의식화 / 개회**
- 시작 직전의 긴장
- 준비된 통제

### start (시작/과정)
- **구조 / 동기화**
- 팀이 하나로 움직이는 순간
- 반복되는 리듬

### summary (요약/종결)
- **종결 / 완결**
- 과정을 끝낸 후의 상태
- 개인이 아닌 집단의 완성

### 금지 장면
- 웃음 위주 컷 ❌
- 가벼운 체험 분위기 ❌
- 개인 영웅 서사 ❌
- 정적인 시설 사진 ❌

### 허용 장면
- 전신을 쓰는 순간 ⭕
- 긴장된 집중 ⭕
- 반복되는 리듬 ⭕
- 통제된 폭발 ⭕

## 10. source_page 코드 (확정)

### 메인 페이지 (6개)
- `home` - Home
- `about` - About
- `programs` - Programs
- `service` - Service 허브
- `track-record` - Track Record
- `contact` - Contact

### Service 하위 페이지 (5개)
- `service--corporate` - 기업·조직
- `service--school-youth` - 학교·청소년
- `service--event` - 스포츠 이벤트·행사
- `service--club-general` - 클럽·일반
- `service--family` - 가족 체험

### 폴백
- `unknown` - 식별 불가 시

### 구현
```javascript
// URL에 source 파라미터 추가
<a href="/contact?source=service--corporate">문의하기</a>

// Contact 폼에 hidden field
<input type="hidden" name="source_page" value="service--corporate" />
```

## 12. HRD 시장 기능 (내부 이해용)

### 로잉프로의 실제 시장 기능
```
로잉프로는 HR 담당자가
위로부터도, 옆으로부터도 욕먹지 않게 만들어주는 동시에,
조직에는 "힘들었지만 납득 가능한 경험"을 남기는
안전한 집단 훈련 장치다.
```

### B2B 고객의 진짜 니즈
1. **무난한 선택** - 실패하지 않는 프로그램
2. **보고 가능성** - 상급자에게 설명 가능
3. **참가자 안전** - 사고 없이 완료
4. **납득 가능한 강도** - 힘들지만 괴롭지 않음
5. **증빙 가능한 전문성** - 국가대표 출신, 13년 업력, 협회
