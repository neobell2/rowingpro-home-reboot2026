CORE VALUES

완벽한 팀워크를 만드는 6가지 핵심 가치.

조화와 협력

여덟 개의 노가 하나의 리듬으로 움직일 때 최고의 속도를 냅니다.

인내와 극기

육체적 한계를 넘어서는 순간, 우리는 정신적으로 성장합니다.

소통과 배려

말 없는 소통이 가장 강력합니다. 서로의 호흡을 느낍니다.

헌신과 신뢰

나보다 팀을 먼저 생각하는 헌신이 승리를 만듭니다.

리듬과 타이밍

0.1초의 오차도 허용하지 않는 정밀한 타이밍의 예술.

목표와 성취

함께 결승선을 통과하는 짜릿한 성취감을 공유합니다.
