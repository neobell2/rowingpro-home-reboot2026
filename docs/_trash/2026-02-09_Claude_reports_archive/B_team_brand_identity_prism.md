## 2. Personality (성격)

### 로잉프로의 성격

| 성격 | 설명 | 표현 방식 |
|---|---|---|
| **진지함** | 조정을 가볍게 만들지 않음 | 차분한 톤, 과장 금지 |
| **검증됨** | 13년 무사고 실적 | 숫자, 팩트 중심 |
| **안전함** | 1보트 1코치 탑승 | 안전 강조, 책임감 |
| **전문적** | 국가대표 출신 | 전문 용어, 체계적 설명 |
| **지속적** | 13년 지속 운영 | 장기 관점, 일관성 |

### Personality 표현

**말투:**
- "우리는 ~합니다" (단언)
- "~을 제공합니다" (명확)
- "검증된 ~" (신뢰)

**금지:**
- "최고의", "최상의" (과장)
- "누구나 쉽게" (전문성 저해)
- "재미있는", "신나는" (레저 연상)

**사용 예:**
```
✅ "전문 코치가 보트에 탑승합니다"
❌ "짜릿한 조정 체험!"

✅ "13년간 무사고 운영"
❌ "최고의 안전 시스템!"

✅ "국가대표 출신 전문가"
❌ "슈퍼스타 코치진!"
```

## 3. Culture (문화/가치)

### 핵심 가치

**불변 정의:**
```
Rowing = Teamwork
```

이것은 슬로건이 아니라 **조정의 정의**

### 문화 구성 요소

**1. 팀워크 전제**
- 조정은 팀워크가 없으면 존재하지 않음
- 개인 영웅 서사 금지
- 집단 동기화 강조

**2. 신뢰 구조**
- 콕스는 앞을 보고
- 로워는 뒤를 보고 전진
- 신뢰는 기능, 감정 아님

**3. 지속 가능성**
- 조정을 지속 가능한 일로 만듦
- 선수·코치의 생계 연결
- 13년 지속 운영 증명

**4. 전문성**
- 체험 아닌 전문 스포츠
- 안전·검증·운영 역량
- 1보트 1코치 원칙

### Culture 전달 방식

**내부:**
- 코치진 교육 시 강조
- 운영 매뉴얼에 명시
- 의사결정 기준으로 사용

**외부:**
- 홈페이지 About 섹션
- B2B 제안서 철학 페이지
- 교육 프로그램 소개

## 4. Relationship (관계)

### 고객과의 관계 방식

**핵심:** 1보트 1코치 동승

| 관계 유형 | 일반 체험업체 | 로잉프로 |
|---|---|---|
| 물리적 거리 | 육지 ↔ 물 위 | **함께 물 위** |
| 관계 깊이 | 일방 지시 | **함께 경험** |
| 책임 범위 | 지시만 | **안전 + 지도** |
| 지속성 | 1회 종료 | 장기 가능 |

### Relationship 특징

**1. 동승 관계**
- 코치가 보트에 탑승
- 같은 공간, 같은 경험
- 즉각 대응, 밀착 지도

**2. 전문가-학습자**
- 국가대표 출신 ↔ 참가자
- 지식 전달 + 경험 공유
- 존중과 신뢰

**3. 장기 파트너십**
- B2B: 반복 계약
- 유학: 수년 훈련
- 가족: 재방문

### Relationship 커뮤니케이션

**접점별 관계 설정:**

| 접점 | 관계 | 톤 |
|---|---|---|
| 첫 문의 | 전문가-의뢰인 | 정중, 명확 |
| 계약 단계 | 파트너 | 신뢰, 투명 |
| 교육 현장 | 코치-참가자 | 밀착, 책임감 |
| 사후 관리 | 장기 파트너 | 지속, 관계 유지 |

## 5. Reflection (외부 인식)

### 고객이 로잉프로를 보는 방식

**타겟별 Reflection:**

#### B2B 의사결정자가 보는 로잉프로
```
"사고 없이 안전한 팀워크 교육을 제공하는
검증된 조정 전문회사"
```

**핵심 이미지:**
- 신뢰할 수 있음
- 검증된 실적
- 전문적 운영

#### 유학 준비 학부모가 보는 로잉프로
```
"아이의 입시 전략을 위해
장기적으로 함께할 수 있는 전문 조직"
```

**핵심 이미지:**
- 체계적
- 지속 가능
- 전략적 파트너

#### 가족 고객이 보는 로잉프로
```
"전문가가 직접 동승하여
안전하게 조정을 경험시켜주는 곳"
```

**핵심 이미지:**
- 안전함
- 전문가가 함께함
- 진짜 조정

### Reflection 관리

**강화할 이미지:**
- 전문회사
- 무사고 실적
- 국가대표 출신
- 13년 지속

**방어할 리스크:**
- 체험업 오해
- 레저 업체 혼동
- 가격 비교 프레임

## 6. Self-image (자아 이미지)

### 고객이 자신을 보는 방식

**로잉프로를 선택한 고객은 자신을 이렇게 본다:**

#### B2B 의사결정자
```
"나는 참가자의 안전을 최우선으로 생각하는
신중한 의사결정자다"
```

**자아 이미지:**
- 책임감 있음
- 신중함
- 검증된 것만 선택

#### 유학 준비 학부모
```
"나는 자녀의 미래를 위해
장기적 관점에서 전략적으로 준비하는 부모다"
```

**자아 이미지:**
- 전략적
- 장기 지향
- 투자 가치 중시

#### 가족 고객
```
"나는 가족의 안전과 새로운 경험을
모두 고려하는 현명한 부모다"
```

**자아 이미지:**
- 안전 중시
- 새로운 것 추구
- 가족 중심

### Self-image 강화 방법

**메시지:**
- "신중한 선택" 칭찬
- "안전을 우선하는" 긍정
- "전략적 결정" 인정

**커뮤니케이션:**
```
✅ "안전을 최우선하시는군요"
✅ "장기적 관점에서 현명한 선택입니다"
✅ "가족을 생각하시는 마음이 느껴집니다"

❌ "저렴하게 드려요"
❌ "누구나 오는 곳"
❌ "쉽고 간단해요"
```

## 6요소 일관성 체크

### 모든 요소가 지향하는 것

```
Physique:   견고한 시각 → 전문성
Personality: 진지한 성격 → 신뢰
Culture:    팀워크 정의 → 본질
Relationship: 동승 관계 → 밀착
Reflection:  전문회사 → 검증됨
Self-image:  신중한 선택 → 안전
```

**일관성:** ✅ 모든 요소가 "전문성 + 안전 + 신뢰"로 수렴
