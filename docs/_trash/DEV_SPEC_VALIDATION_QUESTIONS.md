## 🎯 PHASE 1: 현재 상태 검증 (즉시 답변 필요)

### Q1. 기업·조직 5개 카드 명칭 최종 확정

**현재 상태:**
```
1. 사원교육·조직개발
2. 팀빌딩·협업
3. 리더십
4. 행사·체육대회
5. 건강·웰니스
```

**질문:**
- 설계 단계 보고서(260131)의 이 5개 명칭이 최종인가?
- 변경/수정/추가가 필요한 항목이 있는가?
- 각 명칭의 URL 코드(training, teambuilding, leadership, event, wellness)가 맞는가?

### Q2. Service 하위 나머지 4개 카테고리 구조

**현재 상태:** 미정으로 표시

**질문:**
- 학교·청소년 / 스포츠이벤트·행사 / 클럽·일반 / 가족체험은 각각 어떤 구조를 가질 예정인가?
- 기업·조직처럼 "카테고리 > 5개 카드" 구조를 가질 것인가?
- 아니면 단순 페이지로만 운영할 것인가?
- 현재 미정 상태로 기업·조직만 개발 진입해도 되는가?

### Q3. Service 공통 템플릿 적용 범위

**현재 상태:**
```
10개 섹션 (히어로, 이미지3슬롯, 요약개요, 프로그램개요, 구조운영, 강도난이도, 현장장면, FAQ, PDF버튼, CTA)
```

**질문:**

- 가족체험은 "비상품, 안내 전용"이라고 했는데 같은 템플릿을 쓰는가?

### Q5. 상품 가격 정책 최종 확정

**현재 상태:**

**질문:**

- 패키지 네이밍(SIMPLE/CORE/PRO/MASTER)은 더 이상 사용 안 하는가?
