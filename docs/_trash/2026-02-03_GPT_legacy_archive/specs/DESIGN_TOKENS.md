## 핵심 원칙

### 0. "모든 요소는 의도적이어야 한다"

> **아무 의도 없는 것은 브랜드의 적이다**

웹페이지의 모든 요소는 단 하나도 빠짐없이 의도적으로 사용해야 합니다:
- **왜 이 색상인가?** - 브랜드 아이덴티티, 심리적 효과
- **왜 이 크기인가?** - 시각적 위계, 강조/절제
- **왜 이 위치인가?** - 사용자 시선 흐름, 정보 구조
- **왜 이 폰트인가?** - 가독성, 브랜드 톤앤매너
- **왜 이 비율인가?** - 시각적 균형, 조화
- **왜 이 그리드인가?** - 정보 조직, 반응형 대응

### 1. "Sky Blue 단 하나에서 출발"

**브랜드 철학:**
> "B2B이지만 본질은 스포츠"
> → 생동감, 역동성, 물 위의 스포츠

**컬러 전략:**
- 메인 컬러: Sky Blue **단 하나**
- 블루의 자연스러운 변주 (밝은 Cyan, 짙은 Blue)
- 포인트 컬러: 극히 절제 (Rose, 로고에서 가져옴)
- 이미지: 블루 톤 컬러 그레이딩으로 깊이감 확보

### 2. "블루가 주인공"

**색상 비율:** 블루 2 : 화이트 2 : 다크 2 (균형)
- 블루는 브랜드 아이덴티티 (물, 스마트함, 젊음, 정직함, 유능함)
- 화이트는 호흡 (가독성, 여백)
- 다크는 구조 (Header, Footer만 - 단단함, 끈기)

**주의:** 블랙을 너무 많이 쓰면 "장례회사" 느낌

---

### 1.2 변주 (Sky Blue 계열)

```
Cyan Light:  #22D3EE (Cyan 400)   - 밝은 변주, CTA 전용
Cyan:        #06B6D4 (Cyan 500)   - 중간 변주, CTA 호버
Cyan Dark:   #0284C7 (Sky 600)    - 짙은 변주, 그라디언트
```

**용도:**
- **Cyan Light (CTA 전용)**: Contact 버튼, 주요 전환 지점
- **Cyan (호버)**: CTA 호버 상태, 상호작용 피드백
- **Cyan Dark (변주)**: 블루 그라디언트, 깊이감

**Cyan Light 철학 (CTA 버튼):**
- **역할**: 행동 유도(Call to Action) 전용 색상
- **의미**: "물의 생동감", "행동으로의 초대", "시작의 상쾌함"
- **원칙**:
  - 주요 전환 지점에만 사용 (희소성 = 강조)
  - 페이지당 2-3개로 제한
  - Sky Blue의 밝은 변주 → 브랜드 일관성 유지하면서 눈에 띔

**규칙:** 모두 Sky Blue와 같은 계열, 자연스러운 흐름

## 3. 이미지 전략

### 3.1 실사 이미지

**선택 기준:**
- 물, 조정, 보트 관련 이미지
- **블루 톤 컬러 그레이딩 필수**
- 명암 대비 좋은 이미지

**이유:**
> "블루만 계속 사용하면 지루함 발생
> → 실사 이미지가 블루의 자연스러운 그라데이션 표현
> → 색상 변화만으로는 깨뜨릴 수 없음"

---

### 3.2 오버레이 사용 (필요시만)

**원칙:**
- 텍스트 가독성 위해서만 사용
- 이미지만으로 충분하면 **넣지 않음**
- 전문가가 보정한 이미지는 오버레이 불필요 (World Rowing, Concept2)

**사용 시 고려사항:**
1. 비율 (Opacity): 30-40% 가볍게
2. 배치 (Placement): 텍스트 영역만 부분적으로
3. 투명도: 그라데이션으로 자연스럽게

**목표:** 이미지와 하나로 어우러지게

---

### 3.3 동영상 배경 (추후)

**참고:** Concept2 히어로 = 짧은 에르고미터 사용 영상

**적용 방향:**
- 짧은 조정 영상 (물 위 or 에르고미터)
- 블루 톤 컬러 그레이딩 (편집 시)
- 현재는 이미지로 구조 완성, 나중에 동영상 교체

## 9. 로고 시스템

### 9.1 로고 파일

**파일:** `/public/logo.svg`
**포맷:** SVG (벡터)
**크기:** 8.6KB
**원본:** `Design_ref/로잉프로로고.제미나이수정본.svg`

**의도적 선택:**
- **왜 SVG인가**: 벡터 = 모든 크기에서 선명, 파일 사이즈 최적화
- **왜 투명 배경인가**: Header/Footer 배경과 자연스럽게 조화
- **왜 흰색 요소인가**: Navy Dark/Navy 배경에서 명확한 대비

### 9.2 로고 구조

**구성 요소:**
1. **텍스트**: "ROWING PRO" (흰색 `#FFFFFF`)
2. **보트**: 좌우 블루 삼각형 (블루 `#2B66A3`)
3. **노**: 8개 (상단 4개, 하단 4개, 흰색 `#FFFFFF`)

**의미:**
- 8인승 조정 보트 (8-person rowing boat)
- 블루 = 물, 조정 스포츠
- 블랙/흰색 = 단단함, 명확함

### 9.3 로고 사용

| 위치 | 크기 | 의도 |
|------|------|------|
| **Header** | h-10 (40px) | 브랜드 아이덴티티 강조 |
| **Footer** | h-8 (32px) | 상대적으로 작게, 시각적 위계 |

**호버 효과:**
```astro
class="hover:opacity-90 transition-opacity"
```

**의도:**
- 90% opacity → 클릭 가능함을 암시
- transition → 부드러운 상호작용 경험
- 과하지 않은 피드백

### 9.4 중복 제거

**이전:**
```astro
<img src="/logo.svg" />
<span>ROWING PRO</span>  ← 중복!
```

**현재:**
```astro
<img src="/logo.svg" alt="로잉프로" />  ← 로고만
```

**이유:** 로고 자체에 텍스트 포함, 중복 = 비의도적
